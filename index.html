Function App deployed via Infra-as-Code (ARM)

This project started with adjusting an ARM template to deploy a function app to my Azure sub. I set up two workflows in github actions to automate deployment: The first workflow runs python tests on any push to the master branch. The second workflow deploys the ARM template files to Azure if the tests are succesfull, It uses secrets for my Azure subscription and credential information. If the tests fail, the second workflow will not run and actions feeds back an error message.
